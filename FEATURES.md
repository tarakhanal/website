# Wedding Website Features Overview

## 🎯 What Your Website Includes

### 1. **Landing Page (Animated Envelope)** ✉️
- Beautiful, elegant animated envelope
- Red wax seal in the center
- Tap/click the seal to open the envelope
- Envelope opens with smooth animation
- Countdown timer appears after opening
- "View Details" button to go to main wedding page
- Mobile-optimized for tap interactions

### 2. **Home Page** 🏠
- Welcome message with names and date
- Large countdown timer (Days, Hours, Minutes, Seconds)
- Quick info boxes:
  - Date & Time
  - Location with map placeholder
  - Contact phone number
  - Registry preview
- Call-to-action buttons
- Beautiful gradient background

### 3. **Our Story Page** 📖
- Timeline of your relationship
- Expandable chapters:
  - How We Met
  - First Date
  - Growing Together
  - The Proposal
- Click to expand/collapse each chapter
- Photo placeholders for each section
- Inspirational quote at the bottom

### 4. **About Us Page** 👥
- Individual sections for each person
- Side-by-side layout with photos
- Personal bios
- Fun facts in organized cards
- Favorite quotes and hobbies
- Connects guests to you personally

### 5. **Wedding Timeline Page** ⏰
- Complete wedding day schedule
- Professional timeline design
- Each event includes:
  - Time
  - Title
  - Description
  - Location
- Animated dots on a central line
- "Important Notes" section
- Mobile-friendly alternating layout

### 6. **Photo Gallery** 📸
- Beautiful grid layout (responsive: 1-4 columns)
- 24 photo placeholders ready for images
- Hover effects on desktop
- **Lightbox Features**:
  - Click any photo to open full-screen modal
  - Blurred background behind modal
  - Previous/Next navigation buttons
  - Close button (X)
  - Photo counter (e.g., "5/24")
  - Click background to close
- Smooth zoom animations
- Perfect for showing engagement photos, venue, travel, etc.

### 7. **Reception Music Page** 🎵
- **Song Suggestion System**:
  - Expandable form to add new songs
  - Input fields: Song Title, Artist Name
  - Submit button with validation
- **Voting System**:
  - Each song shows vote count
  - "👍 Vote" button next to each song
  - Prevents duplicate votes per guest (using device storage)
  - Voted buttons appear greyed out after voting
  - Songs auto-sort by vote count (highest first)
- **Song List**:
  - Shows all suggested songs
  - Ranking number (#1, #2, etc.)
  - Artist name
  - Vote count
- **How It Works Section**:
  - Explains the voting process
  - 3-step illustration
- Great for getting guest input on music!

### 8. **Registry Page** 🎁
- Registry information
- Amazon registry button/link
- What you're registered for (checklist)
- FAQ section with answers
- Thank you message
- Reassures guests gifts aren't required
- Beautiful layout with gradient background

### 9. **FAQs Page** ❓
- 12 pre-filled common questions:
  - Arrival time
  - Parking
  - Dress code
  - Plus-ones
  - Dietary options
  - Ceremony location
  - Photo policy
  - Day schedule
  - Alcohol policy
  - RSVP changes
  - Kids welcome?
  - How to help
- Expandable/collapsible Q&A format
- Click to expand each question
- Contact section at bottom
- Phone number linked for calls

### 10. **Navigation** 🧭
- Fixed top navigation bar
- Sticky header (appears on scroll)
- Desktop: All menu items visible
- Mobile: Hamburger menu icon
- Smooth mobile menu animations
- All pages linked
- Logo/initials ("T & B") clickable back to home

## 🎨 Design Features

### Colors & Aesthetics
- **Earthy Boho-Minimalist Theme**
- Warm brown (#8B7355) as primary
- Gold accents (#D4AF85) for highlights
- Sage green, creams, and soft neutrals
- Gradients for subtle depth
- Elegant spacing and padding

### Typography
- **Playfair Display**: Headers (elegant serif)
- **Lora**: Quotes and subheadings (classic serif)
- **Inter**: Body text (clean sans-serif)
- Responsive font sizes (scale with screen)
- Professional line-height and letter-spacing

### Animations & Interactions
- **Framer Motion** animations throughout:
  - Smooth page transitions
  - Hover effects on buttons
  - Image zoom on hover
  - Expandable sections
  - Fade-in animations
  - Staggered animations (elements appear one by one)
- **Micro-interactions**:
  - Button scale on hover/tap
  - Smooth scrolling
  - Modal zoom effect
  - Loading transitions

## 📱 Mobile Optimization

### Mobile-First Design (95% of Users!)
- Touch-friendly button sizes (44px+ minimum)
- Single column layouts on mobile
- Large, readable text
- Hamburger menu instead of full navigation
- Optimized modals for small screens
- Swipe-friendly gallery navigation
- Readable forms and inputs
- Fast loading animations

### Responsive Breakpoints
- Mobile: Up to 640px
- Tablet: 640px - 1024px
- Desktop: 1024px+
- Extra wide: 1280px+
